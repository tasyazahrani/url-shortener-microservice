URL Shortener Microservice

URL Shortener

Example Usage:

Create a Short URL:

POST [project_url]/api/shorturl - with URL body parameter

Access Original URL:

GET [project_url]/api/shorturl/1

Will redirect to:

https://freecodecamp.org/

Example Response:

{
  "original_url": "https://freecodecamp.org",
  "short_url": 1
}
